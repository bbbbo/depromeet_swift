#### 주요 개념

* 정수(integer)
    * 소수점이 없는 수
    * Int
        * 부호가 있는 정수 (signed Integer)
    * UInt
        * 부호가 없는 정수 (unsigned Integer)
  
* 정수 연산
    * 사칙 연산
        * 덧셈 연산 (+)
        * 뺄셈 연산 (-)
        * 곱셈 연산 (*)
        * 나눗셈 연산 (/)
        * 나머지 연산 (%)
    * 복합 대입 연산 (+=, -=, *=, /=, %=)
    * 오버플로/언더플로 연산자 (&+, &-)
  
* 부동소수점수(Floating-Point Number)
